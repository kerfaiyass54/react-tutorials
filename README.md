## 📦 **Installation**

### **Prerequisites**
- **Node.js** (v18.x or later)
- **npm** or **yarn** (v7.x or later)
- **Git** (for cloning the repository)

### **Quick Start**

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/react-tutorials.git
   cd react-tutorials
   ```

2. **Install dependencies**
   ```bash
   # Install all dependencies for all examples
   npm install
   ```

3. **Run a specific example**
   ```bash
   # Navigate to a specific example (e.g., components)
   cd 01-components/react-app-components
   npm install

   # Start the development server
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```





## 🔧 **Configuration**

### **Environment Variables**
Create a `.env` file in each example directory to manage environment-specific settings.

### **Customizing Styles**
Modify the `:root` variables in `index.css` to change the entire theme.

**Example: `index.css`**
```css
:root {
  --primary-color: #646cff;
  --secondary-color: #213547;
  --background-color: #242424;
  --text-color: rgba(255, 255, 255, 0.87);
}
```

### **TypeScript Configuration**
Each example includes a `tsconfig.json` file for TypeScript support.
